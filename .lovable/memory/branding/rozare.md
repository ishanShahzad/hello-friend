# Rozare Brand

The platform is named **Rozare** (formerly Tortrose). The production domain is **rozare.com**.

## Logo
- Modern diamond-bloom mark: 4 overlapping rounded petals (top, right, bottom, left) forming a symmetrical bloom around a white circle with a gradient "R" in the center.
- Gradient: `#FF6B6B` (coral) → `#E94B7C` (magenta) → `#7C3AED` (deep purple).
- Web SVGs: `public/rozare-logo.svg`, `Frontend/public/rozare-logo.svg`, `public/favicon.svg`, `Frontend/public/favicon.svg`.
- Mobile component: `MobileApp/src/components/common/RozareLogo.js` (uses react-native-svg).
- Reference PNG: `src/assets/rozare-mark.png`.

## Naming
- Brand: `Rozare` (capitalized), `rozare` (lowercase for slugs/handles).
- Twitter handle: `@RozareHQ` / `@rozare`.
- Mobile bundle/package: `com.rozare.app`, scheme `rozare://`, applinks `rozare.com`.
- Sentry org/project: `rozare` / `rozare-mobile`.

## Constraint
**Never** reintroduce "Tortrose" anywhere — name, domain, asset filenames, logo, or copy.
